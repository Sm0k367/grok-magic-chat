# Epic Tech AI - Aether Interface

**The flagship chat experience for Epic Tech AI's visionary intelligence.**

Immersive. Intelligent. Epic.

Private real-time interface to Aether — powered by Grok-4 from xAI. Featuring streaming responses, seamless voice interaction (input + output), cosmic visuals, markdown + code highlighting, and a thoughtfully crafted experience for innovators, researchers, and builders.

## Features

- **Aether**: Our flagship AI persona — visionary, technically profound, witty, and inspiring
- Real-time token streaming (watch thoughts form in real time)
- Voice input that auto-transcribes and submits
- AI voice output (text-to-speech on responses)
- Beautiful cosmic UI with glassmorphism, neon accents, and epic gradients
- Sidebar with prompt inspiration for breakthrough tech ideas
- Custom system prompt editor
- Full markdown and syntax-highlighted code support
- Built with Next.js 14, Tailwind, and the xAI API

## Quick Start

```bash
git clone https://github.com/Sm0k367/grok-magic-chat.git
cd grok-magic-chat
npm install
cp .env.example .env.local
# Add your XAI_API_KEY from https://console.x.ai/
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and begin your epic journey.

## Environment Variables

Create `.env.local`:

```env
XAI_API_KEY=xai-your-key-here
```

Get your key at [console.x.ai](https://console.x.ai/).

# Grok Magic Chat

A beautiful, real-time chat interface powered by Grok-4 from xAI.

## Features
- Real Grok-4 responses
- Clean, modern UI with Tailwind
- API route backend
- Ready for Vercel deployment

## Environment Variables
Create a `.env.local` file with:
```env
XAI_API_KEY=your_xai_api_key_here
```

## Deploy on Vercel
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com/new)
3. Import the repo
4. Add `XAI_API_KEY` in Environment Variables
5. Deploy

Live in seconds.

Made with ❤️ using the xAI SDK.
